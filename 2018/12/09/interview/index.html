HTML

  • and 还没想到

CSS

  • 行内元素、块元素
  • 盒模型
  • 选择器及优先级（id/class/tag 等）
  • 伪类
  • display 的值及作用
  • posiiton 可选值及其区别
  • px em rem 的区别

JS

  • 参考图谱

网络

  • TCP 协议、三次握手
  • http https http2.0
  • 加密：对称、非对称
  • 还有一道题想不起来知识点了，等我想想

数据结构

  • 排序：冒泡、快排、归并、堆等等（时间复杂度和空间复杂度）
  • 查找：二分查找等
  • 队列：队列数据结构的实现，及基础删除、插入、查找等算法（单向、双向）
  • 栈：数据结构的实现，及基础删除、插入、查找等算法
  • 二叉树：前中后序（递归于非递归的实现方法）
  • 图、树：深度、广度优先搜索等

vue/react/node等

  • vue、react 会根据工作中是否用到来问，对自己用的熟悉掌握即可
  • 发包、单测、配置命令等
  • 优化
